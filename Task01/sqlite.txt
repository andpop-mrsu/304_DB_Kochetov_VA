Режимы вывода данных (output modes) в утилите sqlite3:
- ascii     - Столбцы/строки разделены символами 0x1F и 0x1E
- box       - Таблицы с использованием символов рисования Unicode
- csv       - Значения, разделённые запятыми
- column    - Вывод в столбцах
- html      - HTML-код <table>
- insert    - SQL INSERT-операторы для таблицы
- json      - Результаты в формате JSON-массива
- line      - Одно значение на строку
- list      - Значения, разделённые символом "|"
- markdown  - Формат таблицы Markdown
- qbox      - Сокращение для "box --wrap 60 --quote"
- quote     - Экранирование ответов как SQL-литералов
- table     - ASCII-арт таблица
- tabs      - Значения, разделённые табуляцией
- tcl       - Элементы TCL-списка

Файлы, необходимые для работы SQLite:
- sqlite3.exe       - Основной консольный клиент (интерпретатор команд)
- sqldiff.exe       - Утилита для сравнения баз данных
- sqlite3_analyzer.exe - Утилита для анализа структуры БД
- sqlite3_rsync.exe - Утилита для синхронизации баз данных

SQLite является встраиваемой СУБД и хранит всю базу данных в одном файле.
Для работы не требуется отдельный сервер - достаточно только sqlite3.exe.
